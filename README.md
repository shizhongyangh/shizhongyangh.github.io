## ⚡ 快速上手

### 环境准备
```bash
# 安装Hexo脚手架
npm install hexo-cli -g
```

### 本地开发
```bash
# 克隆本仓库
git clone https://github.com/shizhongyangh/shizhongyangh.github.io.git

# 安装依赖
npm install

# 启动本地服务器
hexo server
```

## 🙏 项目渊源
本项目基于 [fomalhaut1998] 的开源项目 [hexo-theme-Fomalhaut]([https://github.com/example/example](https://github.com/fomalhaut1998/hexo-theme-Fomalhaut)) 二次开发，主要改进包括：
- ✅ 升级至Hexo 6.x版本
- ✅ 增加暗黑模式切换
- ✅ 优化移动端阅读体验
- ✅ 集成Gitalk评论系统
